ZIP Naming Rule
Define the filename format for the downloaded ZIP archive. You can choose a preset or type a custom name.

Available variables:
  • {repo} : Repository name
  • {branch} : Branch name
  • {owner} : Repository owner username
  • {path} : Selected path/folder name
  • {date} : Current date (YYYY-MM-DD)
  • {datetime} : Full timestamp (YYYYMMDD-HHMMSS)
  • {ts} : Short date (YYYYMMDD)

Custom Token

GitHub API Access
Choose how GitZip Pro accesses GitHub API:
  • Anonymous Access: Uses public API without authentication (rate limited to 60 requests per hour)
  • Custom Token: Use your personal GitHub token for higher rate limits and access to private repositories
⚠️ Anonymous Mode Limitation

Limitations: Only 60 requests per hour shared with all other users on your network. This is very likely to be exhausted quickly.

Recommendation: Create a free Personal Access Token to get 5000 requests per hour, and access your private repositories.

GitHub Personal Access Token
You can either manually enter your GitHub Personal Access Token, or click the authorization buttons below to get a token automatically via OAuth.

Authorization options:
  • Public Repos Only: Grants access to public repositories only
  • Full Access: Grants access to both public and private repositories

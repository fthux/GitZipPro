  • Custom Token: Use your personal GitHub token for higher rate limits and access to private repositories
GitHub Personal Access Token
Enter your GitHub Personal Access Token (PAT).

How to create a token:
  1. Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token" → "Generate new token (classic)"
  3. Give it a descriptive name (e.g., "GitZip Pro")
  4. Select scopes: repo (for private repos) and read:org (optional)
  5. Click "Generate token" and copy the token
Note: The token is stored locally in your browser and never sent to any server except GitHub.

GitZip Pro 是一款开源的浏览器扩展，用于从 GitHub 仓库中选择并下载指定的文件和文件夹。

官方网站：
https://gitzippro.fthux.com

GitZip Pro 的开发灵感来自 GitZip for GitHub，并向它最初提出的优秀理念致敬：让用户可以轻松下载 GitHub 仓库中真正需要的部分。

在此基础上，GitZip Pro 提供了更加完整的“Pro”级下载体验，包括私有仓库支持、浏览器本地 ZIP 打包、下载历史与统计、智能忽略规则、文件大小显示、自定义文件名、界面个性化以及更多工作流程增强功能。

只下载你真正需要的内容

GitZip Pro 会在 GitHub 仓库文件列表中添加选择框。选择一个或多个文件和文件夹后，即可将它们下载为 ZIP 压缩包，同时保留仓库原有的目录结构。

你也可以通过浏览器右键菜单快速下载单个文件或文件夹，无需克隆或下载整个仓库。

PRO 功能

• 直接在 GitHub 页面选择多个文件和文件夹
• 将选中的内容下载为 ZIP 压缩包
• 保留仓库原有的目录结构
• 通过右键菜单下载单个文件或文件夹
• 直接在浏览器本地创建 ZIP 压缩包
• 匿名访问和下载公开仓库
• 可选择连接 GitHub，以提高 API 限额并访问私有仓库
• 使用仓库名、分支、路径和日期等变量自定义 ZIP 文件名
• 使用内置预设或自定义忽略规则排除不需要的文件
• 可选择在仓库页面显示文件大小
• 查看存储在本地的下载历史和统计信息
• 配置下载完成通知和可选提示音
• 自定义主题、强调色和下载按钮位置
• 遇到临时网络错误或 GitHub API 限流时自动重试
• 自动适配 GitHub 页面导航，无需手动刷新页面

公开仓库与私有仓库

公开仓库支持匿名下载，无需连接 GitHub 账号。

如需提高 GitHub API 请求限额或访问私有仓库，可以选择连接 GitHub 账号，或者在扩展设置中提供兼容的 GitHub Access Token。

隐私说明

仓库文件直接从 GitHub API 获取，并在浏览器本地打包为 ZIP 压缩包。GitZip Pro 不会将下载的仓库内容上传至自己的服务器。

扩展设置、访问凭据、下载历史和统计信息通过 Chrome 扩展存储保存在本地。只有当你主动选择连接 GitHub 账号时，扩展才会访问 GitZip Pro 授权服务。

使用方法

1. 打开受支持的 GitHub 仓库页面。
2. 选择需要下载的文件和文件夹。
3. 点击 GitZip Pro 下载按钮。
4. 等待 ZIP 压缩包创建并保存完成。
